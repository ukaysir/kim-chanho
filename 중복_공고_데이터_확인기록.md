# 김해행정마찰지도 P0·P1 확인 기록

- 기준일: 2026-07-15
- 조사 범위: 김해시·국민생각함의 공개 공고, 공개 업무분장, 공개 성과보고서, 공개 사업자료
- 조사 한계: 전화·이메일 질의와 내부 시스템 조회는 수행할 수 없으므로, 공개자료에서 찾지 못한 사실을 "없음"으로 판정하지 않는다.
- 현재 판정: **조건부 진행**. 공개자료상 동일 사업은 확인되지 않았으나, 역대 채택·시행 사업 전체와 내부 데이터 제공 가능성은 담당 부서 확인 전까지 `미확인`이다.

## 1. 핵심 결론

1. 원 공고의 접수기간은 2026-06-22부터 2026-07-21 18:00까지이며, 1인 또는 1팀당 1건만 제출할 수 있다. 동일·유사 제안은 먼저 접수된 제안을 우선하고, 이미 채택·시행 중이거나 기본 구상이 유사한 제안은 심사 대상에서 제외될 수 있다.
2. 2026-07-15 기준 공개 검색에서는 해당 공고의 정정·연장 공지를 찾지 못했다. 이는 `미발견`일 뿐 `정정 없음 확인`이 아니므로 정책기획과에 최종 확인해야 한다.
3. 공개된 2025년 김해시 정책제안 수상작 12건에는 검색어와 콜센터 이관 신호를 결합해 월별 행정 마찰을 개선하는 동일 제안이 보이지 않았다. 다만 이 확인 범위는 공개된 2025년 수상작뿐이며, 역대 채택작·부서 자체사업·미수상 시행안은 포함하지 않는다.
4. 전국 공개 제안에는 AI 민원상담과 민원 자동분류·배정 아이디어가 이미 존재한다. 따라서 본 제안의 차별점은 "AI 상담 시스템 신설"이 아니라 **민원 발생 전 검색 신호 + 콜센터 전화돌림 신호 + 담당자의 월별 검증 + 실제 메뉴·문구·연결표 수정 + 전후 비교**로 고정해야 한다.
5. 검색 채널은 인기 검색어와 검색량이 공개되어 있어 최소형 파일럿의 입력 신호로 쓸 수 있다. 클릭, 재검색, 검색결과 0건 여부와 원시 로그 제공 가능성은 공개자료로 확인되지 않았다.
6. 콜센터는 2024년 성과보고서에서 총응답과 전화돌림을 바탕으로 자체처리율을 월별 측정한 사실이 확인된다. 전체 전화돌림 집계는 존재하지만 문의유형, 최초 이관, 재이관, 상담 메모 등의 세부 필드는 확인되지 않았다. `38.4%`는 전체 전화돌림 비중으로 해석할 수 있을 뿐, 오이관율로 쓰면 안 된다.
7. 현재 공개 업무분장 기준 총괄 후보는 AI정책과 데이터융합팀, AI 적용 후보는 AI전략팀이다. 홈페이지 운영은 정보통신과 정보기획팀, 개인정보·보안 검토는 정보보호팀, 콜센터 운영 창구는 소통공보관 시민소통팀으로 보는 것이 타당하다. 이는 공식 사업 배정이 아닌 업무분장 기반 후보 지정이다.

### 3.3 전국 공개 유사 제안과 차별화

공개된 유사 제안은 주로 AI 챗봇, 상담 자동화, 민원 자동분류·담당부서 배정에 초점을 둔다. 본 제안은 아래 범위를 벗어나지 않아야 차별성이 유지된다.

- 신규 범용 민원상담 플랫폼을 구축하지 않는다.
- 공개 검색어와 기존 콜센터 집계를 먼저 쓰고, 확인되지 않은 원시 로그를 필수 전제로 삼지 않는다.
- 월별 상위 마찰 후보를 담당자가 검증한다.
- 개선 대상은 홈페이지 메뉴·안내문·동의어·연결표·상담 매뉴얼 등 작은 운영 변경으로 제한한다.
- 같은 지표를 개선 전후에 다시 측정해 변경 효과를 남긴다.

### 4.3 익명 집계 원칙

- 검색어에도 이름, 전화번호, 주소 등 개인정보가 포함될 수 있으므로 원문 전체를 제안팀에 전달하지 않는다.
- 월별 집계, 표준화된 주제 분류, 소수빈도 억제 기준을 적용한다.
- 상담 메모와 녹취는 최소형 파일럿에서 제외한다.
- 부서·유형별 건수는 개인을 역추적할 수 없는 수준으로만 제공한다.
- 원시 데이터 보관과 접근권한은 기존 시스템 담당 부서가 유지하고, 파일럿에는 가공 집계만 제공한다.
- 실제 기준은 정보보호팀의 개인정보·보안 검토 후 확정한다.

## 6. 유지보수 범위와 비용 확인

공개자료만으로 현재 계약의 과업범위와 추가 변경단가를 확인하지 못했으므로 총액을 임의 산정하지 않는다. 아래 네 항목의 답변을 받은 뒤 내부 인건비와 외부비를 나눠 산정한다.

| 비용 항목 | 최소형 범위 | 확인 질문 |
|---|---|---|
| 데이터 추출·익명화 | 월 1회 집계 파일 | 기존 운영업무에 포함되는가, 별도 개발이면 예상 공수·단가는 얼마인가 |
| 분석·검증 | 상위 후보 분류와 담당부서 회의 | AI정책과 내부 분석으로 가능한가, 기존 데이터 분석 사업 성과물을 재사용할 수 있는가 |
| 홈페이지 변경 | 메뉴, 안내문, 검색동의어, 연결 링크 소규모 수정 | 현 유지보수 계약에 포함되는 변경 건수와 제외 범위는 무엇인가 |
| 콜센터 변경 | 연결표와 상담 매뉴얼 수정 | 현 위탁·유지보수 범위에 포함되는가, 시스템 필드 변경이 필요한가 |
| 선택형 AI 기능 | 익명 집계의 주제 분류·요약 | 범정부 AI 공통기반을 활용할 수 있는가, 별도 API·개발비가 발생하는가 |

비용을 낮추는 기본안:

- 신규 앱이나 대시보드를 만들지 않고 기존 문서·스프레드시트·업무보고 형식을 사용한다.
- 클릭·재검색·재이관 필드가 없으면 새 수집기능을 먼저 개발하지 않고 공개 검색량과 기존 전화돌림 집계로 시작한다.
- 12주 동안 3~5개 마찰 항목만 선정한다.
- 변경은 기존 유지보수 범위 안의 메뉴·문구·동의어·연결표 수정부터 시행한다.
